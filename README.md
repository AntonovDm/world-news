# World News

Проект на этапе разработки.

Демо можно посмотреть здесь → [Demo](https://all-world-news.vercel.app/)

## Текущий stack

- React
- TypeScript
- SCSS module

### Будет внедрено:

- React Router
- Redux Toolkit
- OpenWeatherAPI
- Новый дизайн для веб-приложения и телефона.
- Темная тема.
- Авторизация с сохранением новостей и избранным.
